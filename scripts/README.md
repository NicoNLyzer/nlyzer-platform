# NLyzer Scripts - Operational Utilities

This directory contains operational scripts for managing the NLyzer platform.

## Scripts

### Data Ingestion
- Scripts for bulk data import from various e-commerce platforms
- ETL pipelines for customer data synchronization

### Deployment
- GCP infrastructure provisioning scripts
- Database migration utilities
- Backup and recovery procedures

### Monitoring
- Health check scripts
- Performance benchmarking tools
- Log aggregation utilities

## Usage
All scripts should be run from the project root directory.
Most scripts require appropriate environment variables to be set.

For detailed architecture, see [The Unified Architectural Blueprint](../docs/UNIFIED_ARCHITECTURAL_BLUEPRINT.md).
